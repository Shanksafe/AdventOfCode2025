# Day 6
## Summary

## Part 1
The worksheet is processed column by column. Each vertical block represents a problem.
First extracts numbers and then the operator from each block, evaluate the result and sum all results to get the grand total.

## Part 2

Part 2 changes how the worksheet is read. Problems must be processed
from right to left and each vertical number must be read bottom to top. 
The solution scans problem blocks from the right side of the worksheet then reverses each extracted number string.
