# Day 1
## Summary

## Part 1

This problem simulates movement on a circular dial (0–99) based on a list of left and right rotations. 
Starting from position 50, each instruction updates the position to wrap around the dial.
The goal is to count how many times the dial lands exactly on 0.

## Part 2
Counts every time the dial passes through 0 during rotations, not just at the end.
