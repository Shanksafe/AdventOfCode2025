# Day 2
## Summary

## Part 1

This problem involves a long input of number ranges where the goal is to
separate invalid IDs from valid ones. Then summing up the total invalid IDs to solve the problem. 

## Part 2

Part 2 extends the rule so that an ID is invalid if it is made from any digit pattern repeated two or more times (not just twice).

Instead of splitting the number into two halves, the solution tests all possible pattern lengths and checks whether
repeating that pattern reconstructs the original number.
